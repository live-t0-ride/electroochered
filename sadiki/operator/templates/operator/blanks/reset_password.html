учётной записи системы «Электроочередь»

  1. Учетная запись системы электронной очереди в детские сады «Электроочередь» выдаётся Заявителю для доступа к Личному кабинету.   2. Регистрационное имя и пароль используются для авторизации доступа к системе и учёта действий, выполняемых Заявителем.
  3. Пароль учетной записи является конфиденциальной информацией и должен быть известен только тому лицу, для которого создана учётная запись.
  4. При создании учётной записи оператором системы устанавливается временный пароль. После получения этого пароля Заявитель входит в систему, самостоятельно устанавливает рабочий пароль.
  5. Запрещается выполнение любых действий в системе «Электроочередь» с использованием временного пароля, кроме установки рабочего пароля.
  6. Восстановление пароля учётной записи самостоятельно пользователем невозможно. При утере пароля для его восстановления необходимо обратиться к оператору системы. Для существующей учётной записи будет установлен временный пароль (см. п.4).
  7. Заявитель несёт личную ответственность за все действия, выполняемые в системе «Электроочередь», с использованием переданной ему учётной записи.
